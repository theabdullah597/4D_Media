# 4D Media - Custom Printing Platform

A complete full-stack e-commerce platform for custom print-on-demand merchandise. Users can design personalized products (T-shirts, mugs, hoodies, caps) and place orders online.

## Features

### Customer Features
- **Multi-View Designer**: Customize Front, Back, and Sleeves independently.
- **Bulk Pricing Engine**: Live tiered discounts for large orders.
- **Secure Accounts**: User registration, login, and order history.
- **UK Localization**: GBP (£) pricing and UK postcode validation.
- **WhatsApp Integration**: Direct support link for verified orders.

### Admin Features
- **Interactive Analytics**: Real-time revenue and product performance charts.
- **Granular Order Inspection**: View designs per surface (e.g., "Left Sleeve").
- **Product Management**: Robust variant and price tier control.
- **Order Management**: Status tracking and GBP financial reporting.

## Technology Stack

- **Frontend**: React 18 + Vite + Recharts + Framer Motion
- **Backend**: Node.js + Express + SQLite
- **Auth**: JWT (User & Admin scopes)
- **Validation**: Custom UK Postcode Logic

## Future Enhancements

- Advanced Image Filters (AI-based background removal)
